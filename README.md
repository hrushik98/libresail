## how to use
### Bare-metal
- clone this repo
- install requirements with: `pip install -r requirements.txt`
- run dev server with: `flask --app main run`
- view app by visiting in browser: `http://127.0.0.1:5000`

### docker
- run: `docker compose up -d`
- view app by visiting in browser: `http://127.0.0.1:5000`
